# CLRvoyance - Shellcode 生成器

一个现代化的 Web 工具，用于生成在非托管进程中加载 .NET 托管程序集的 Shellcode。

## ✨ 亮点特性

### 🌐 在线使用
- **纯前端实现**：无需安装任何软件，直接在浏览器中打开即可使用
- **本地处理**：所有操作在本地完成，文件不会上传到服务器，保护隐私安全
- **跨平台支持**：支持 Windows、macOS、Linux 等所有现代浏览器

### 🔄 转换功能
- **多种输出格式**：
  - 二进制文件 (.bin) - 原始 shellcode 二进制数据
  - C 语言头文件 (.h) - 可直接嵌入 C/C++ 项目
  - C# 字节数组 (.cs) - 可直接用于 .NET 项目
- **智能格式转换**：自动将 shellcode 转换为不同编程语言所需的格式

## 🚀 功能特性

- ✅ 支持 32 位和 64 位平台
- ✅ 支持多种加载模式（普通、APC、新域）
- ✅ 拖拽上传文件，操作便捷
- ✅ 现代化暗黑风格 UI
- ✅ 实时生成，即时下载



### 2. 使用工具

1. **打开文件**：在浏览器中打开 `clrvoyance.html`
2. **上传程序集**：
   - 拖拽 `.exe` 或 `.dll` 文件到上传区域
   - 或点击上传区域选择文件
3. **选择配置**：
   - 平台架构：32 位或 64 位
   - 加载模式：APC 模式、新域模式
   - 输出格式：二进制、C 语言、C#
4. **生成下载**：点击"生成 Shellcode"按钮，文件将自动下载

## 📝 输出格式说明

### 二进制格式 (.bin)
原始 shellcode 二进制数据，可直接用于内存执行。

### C 语言格式 (.h)
生成 C 语言风格的十六进制数组，示例：
```c
"\x61\x72\x69\x61\x6e\x74\x49\x6e\x69\x74\x00\x4d\x5a\x90\x00\x03"
"\x00\x00\x00\x00\x00\x00\x00\x00\x00\x00\x00\x00\x00\x00\x00\x00"
...
```

### C# 格式 (.cs)
生成 C# 字节数组，示例：
```csharp
byte[] shellcode={
0x61,0x72,0x69,0x61,0x6e,0x74,0x49,0x6e,0x69,0x74,0x00,0x4d,0x5a,0x90,0x00,0x03,
0x00,0x00,0x00,0x00,0x00,0x00,0x00,0x00,0x00,0x00,0x00,0x00,0x00,0x00,0x00,0x00,
...
};
```
